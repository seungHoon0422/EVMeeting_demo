## 🌲 3. Gitlab Conventions

#### branch tree

- master
  - develop
    - feature-chat
      - FE-chat
      - BE-chat
    - feature-accounts
      - FE-accounts
      - BE-accounts
    - feature-meeting
      - FE-meeting
      - BE-meeting

#### Commit Convention

- Fix : 잘못된 동작을 고칠 때

  > Fix typo in Home.vue

- Add : 새로운 것을 추가할 때

  > Add Detail.vue

- Remove : 삭제가 있을 때

  > Remove Detail.vue

- Update : 정상적으로 동작하는 파일을 보완하는 경우

  > Update login logic to accounts.js
